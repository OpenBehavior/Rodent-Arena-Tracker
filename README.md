# Rodent-Arena-Tracker

A closed-loop system for machine vision rodent tracking and task control.

Designed by Jonathan Krynitsky (alexxai@wustl.edu)

RAT is a low-cost system to track and report mouse activity while they are contained in a simple arena. The system uses an OpenMV M7 microcontroller to acquire images of the arena and process them in real-time to report mouse centroid data to a text file. 

Original posting: https://hackaday.io/project/162481-rodent-arena-tracker-rat
